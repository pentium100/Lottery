Extension to Ext.form.ComboBox [2.x], by Andrei Neculau

ExtJS Forum | Testcase source

  • There is no valueNotFoundText displayed when trying to select a non-existing value
  • Values are kept internally in an array
  • Introduces the ability to erase the value by using the keyboard
  • The change event will fire upon forceSelection:false and bluring as well.
  • The change event will fire upon selection also. This is my understanding of QuirksMode explanation.

  • IMPORTANT! Contains a "JS hack" for Ext.DataView - changes deselect method to support surpressEvent parameter
  • IMPORTANT! Contains a "CSS hack" in order to add a margin of 1px to listed items
  • IMPORTANT! Updates to "value" text fields are done by listeners to CHANGE event
  • IMPORTANT! When history capabilities are enabled, the store will suffer changes
  • multiSelect: bool - single or multiple selection (default:false)
  • minLength: int - minimum number of elements to have selected
  • minLengthText: string - text to display when minLength condition is not satisfied
  • maxLength: int - maximum number of elements to have selected
  • maxLengthText: string - text to display when maxLength condition is not satisfied
  • history: bool - act as history field; only if mode is 'local' (remember entered values to the top of the list)
  • historyMaxLength: int - number of entered values to remember, 0 means remember all (defaults to 0)
  • separator: string - separator to use for the values passed to setValue (defaults to comma)
  • displaySeparator: string - separator to use for displaying the values (defaults to comma)
Methods
  • addValue: (string) - add a value and select it in the list
  • removeValue: (string) - remove a value and deselect it in the list
  • addHistory: (string) - add a value in field's history
  • getValue: (bool) - return the value as string or as array
  • getRawValue: (bool) - return the raw value as string or as array

Properties
  • valueArray - array denoting the selected values Should be considered private
  • rawValueArray - array denoting the selected raw values Should be considered private
